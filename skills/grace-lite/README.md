# grace-lite

GRACE-lite (Graph-RAG Anchored Code Engineering, lite variant) adds **semantic
code markup** to a project — region-based contracts and anchors that make code
machine-navigable.

## Usage

This is an **agent skill**. Install it. Then ask the agent to run **GRACE init**
or **GRACE review** or ask questions about GRACE markup.
